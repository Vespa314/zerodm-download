zerodm-download-
================

自动获取ZERODM下载链接

修改spyder/zerodm.py文件里面的starturl可以一口气获取多部动漫链接

需要scrapy库支持

================
添加非scrapy版本，实际上效果更好。。
运行：noscrapy.py

================
添加Mathematica版本
参见：mathematica version.txt

================
添加PHP版本
参见：zerodm.php
================
